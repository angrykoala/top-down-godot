# top-down-godot
A simple top-down shooter for learning godot

* Node vs. Scene
* Input
* Ready, process and physics process
* Raycasting (hitscan weapons)
* Groups
* Signals
* PackedScenes
* Line2D
* Sound
* Light
* Tilemap
* KineticBody/physics (a bit)
